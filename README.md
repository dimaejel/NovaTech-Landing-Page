# NovaTech AI Landing Page 🚀

A modern and responsive landing page built for Codveda Front-End Development Internship Task.

## 📌 About The Project

NovaTech AI is a technology company landing page that presents digital services including web development, AI solutions, and UI/UX design.

The goal of this project is to create a responsive and interactive website using HTML, CSS, and JavaScript.

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Git & GitHub

## ✨ Features

- Responsive design for mobile, tablet, and desktop
- Modern navigation bar
- Mobile hamburger menu
- Smooth scrolling navigation
- Hero section with call-to-action button
- Services cards
- About section
- Contact section
- Basic animations and hover effects
